# הוספת תמונות לגלריה

## מה קרה?
הגלריה היתה מנסה לטעון תמונות מאתרים חיצוניים (unsplash), אך התמונות שלך נמצאות בגוגל דרייב.
הפתרון: להשתמש בתמונות מקומיות שאתה מורידה לפרויקט.

## איך להוסיף תמונות (3 שלבים פשוטים)

### שלב 1: המנייה כבר קיימת
תיקיית תמונות נוצרה כבר ב:
```
client/src/assets/gallery/
```

### שלב 2: הורד וארגן את התמונות
1. לך לקישור: https://drive.google.com/drive/folders/1QlrpDy4-vvzVBoZAgy0vZ_8Sif9yvUfh
2. הורד את התמונות
3. שנה את שמות הקבצים לפי הקטגוריה:
   - תמונות ילדים: `kids-1.jpg`, `kids-2.jpg`, `kids-3.jpg`
   - תמונות משפחה: `family-1.jpg`, `family-2.jpg`, `family-3.jpg`
   - תמונות תינוק: `newborn-1.jpg`, `newborn-2.jpg`
   - תמונות פעוט: `toddler-1.jpg`, `toddler-2.jpg`

4. העתק את התמונות לתיקייה: `client/src/assets/gallery/`

### שלב 3: הגלריה כבר מוכנה
קובץ `Gallery.jsx` כבר מעודכן ומחפש את התמונות בשמות הנכונים.

אם רוצה לשנות כותרות או להוסיף תמונות נוסxxxxxערוך את:
```
client/src/components/Gallery.jsx
```

## מבנה התיקייה
```
client/
└── src/
    └── assets/
        └── gallery/          ← הוסף תמונות כאן
            ├── kids-1.jpg    (הוסף את התמונות שלך)
            ├── kids-2.jpg
            ├── family-1.jpg
            └── ...
```

## בדיקה
כשתוסיף תמונות:
1. רענן את הדף בדפדפן (Ctrl+R)
2. אתה אמור לראות את התמונות שלך בגלריה עם:
   - אפקט טשטוש בתמונות
   - כפתורי סינון לפי קטגוריה
   - אפקט זהב על גבול התמונה בעת ריחוף
   - כפתור "View" בעת ריחוף
